A responsive and interactive **Student Registration System** developed using **HTML, CSS, and JavaScript (DOM Manipulation)**.  
This project was created as part of an academic assignment to demonstrate front-end development skills such as **form handling**, **validation**, **DOM manipulation**, and **localStorage-based data persistence**.

---

##  Features

-  **Add, Edit, Delete Student Records**
-  **Persistent Storage** using `localStorage` (data doesnâ€™t disappear on refresh)
-  **Search Functionality** (by name or student ID)
- **Form Validation**
- Name: alphabets only  
- Student ID: numeric only  
- Email: valid email format  
- Contact Number: at least 10 digits
-  **Dynamic Scrollbar** appears automatically when records exceed six entries
-  **Modern Glass-Effect UI** with soft colors and smooth transitions
-  **Fully Responsive Design** for mobile, tablet, and desktop screens

| Technology | Purpose |
